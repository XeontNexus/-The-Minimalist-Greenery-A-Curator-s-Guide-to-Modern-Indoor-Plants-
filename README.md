# The Minimalist Greenery: A Curator's Guide to Modern Indoor Plants

## Konsep Utama
**"Less is More, Green is Must"**

Portfolio web yang berfokus pada kurasi tanaman indoor dengan siluet geometris yang cocok dengan arsitektur modern.

## Fitur Unggulan

### 🏠 Space Matcher
Fitur pencocokan ruangan yang merekomendasikan tanaman berdasarkan:
- Kamar Mandi (kelembapan tinggi, cahaya rendah)
- Meja Kerja (cahaya sedang, ruang terbatas)
- Ruang Tamu (cahaya baik, ruang luas)

### 🌿 The Curated List
Katalog tanaman dengan fitur filtering:
- Cahaya Rendah
- Mudah Dirawat
- Menyaring Udara

### 🔬 Anatomy of a Leaf
Detail makro daun dengan deskripsi teknis manfaat bagi udara

### 💧 Care Lab
Kalkulator kebutuhan perawatan berdasarkan:
- Suhu ruangan
- Kelembapan
- Intensitas cahaya

### 🛒 The Shop
Koleksi eksklusif dengan sistem pembelian via email

### 💬 Community Thoughts
Sistem komentar dengan validasi email

## Teknologi yang Digunakan

- **HTML5** - Struktur semantik
- **Tailwind CSS** - Desain bersih & responsif
- **Vanilla JavaScript** - Interaktivitas
- **Font Awesome** - Ikon
- **Google Fonts** - Typography (Playfair Display + Inter)

## Palet Warna

- `#F9F9F9` - Off-white/Dinding Minimalis
- `#2C3E50` - Charcoal/Pot Hitam
- `#4A5D4E` - Deep Leaf Green
- `#E0C9A6` - Light Oak/Elemen Kayu

## Typography

- **Playfair Display** - Font Serif untuk judul (kesan elegan)
- **Inter** - Font Sans-serif untuk isi (kesan modern)

## Struktur Halaman

1. **The Entrance (Home)** - Animasi slow-zoom tanaman dalam pot keramik putih
2. **Space Matcher** - Pencocokan ruangan dengan rekomendasi tanaman
3. **The Curated List** - Katalog tanaman dengan filtering
4. **Anatomy of a Leaf** - Detail makro dan manfaat teknis
5. **Care Lab** - Kalkulator kebutuhan perawatan
6. **The Shop** - Galeri dan sistem pembelian
7. **Community Thoughts** - Sistem komentar

## Tanaman dengan Karakter Minimalis

- **Sansevieria (Snake Plant)** - Garis vertikal tegas
- **Fiddle Leaf Fig** - Daun besar sebagai statement piece
- **Zamioculcas (ZZ Plant)** - Daun mengkilap, sangat tahan banting
- **Monstera Adansonii** - Kesan tropis tapi tetap rapi
- **Pothos** - Tanaman merambat fleksibel
- **Rubber Tree** - Daun lebar mengkilap

## Cara Menjalankan

1. Buka `index.html` di browser modern
2. Atau jalankan dengan server lokal:
   ```bash
   # Menggunakan Python
   python -m http.server 8000
   
   # Menggunakan Node.js
   npx serve .
   
   # Menggunakan PHP (karena menggunakan XAMPP)
   php -S localhost:8000
   ```

3. Akses `http://localhost:8000`
